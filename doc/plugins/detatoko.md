Detatoko
=======

ゲーム専用プラグイン第1弾、「でたとこサーガ」用プラグイン。
でたとこサーガ公式サイト: http://detatoko-saga.com/

コマンド
--------

### スキルランクから判定値を算出 (`.ds<SR>`)

`<SR>` にスキルランクを入れると、ダイスを振って判定値を求めます。  
追加で一つだけ、固定値を足すことができます。

#### 例

```
.ds3
> でたとこサーガ [koi-chan]<判定値>: スキルレベル = 3 -> [2, 5, 3, 2] = 8
.ds3+3
> でたとこサーガ [koi-chan]<判定値>: スキルレベル = 3 -> [3, 5, 6, 6] + 3 = 15
```

### 体力烙印の獲得 (`.dvst`)

体力烙印を獲得します。
最初に1ゾロが出た時の振り足しなどにも対応しています。

#### 例

```
.dvst
> でたとこサーガ [koi-chan]<体力烙印>: [5, 2] -> 7:【衝撃】
```

### 気力烙印の獲得 (`.dmst`)

上と同様に気力烙印を獲得します。

#### 例

```
.dmst
> でたとこサーガ [koi-chan]<気力烙印>: [3, 2] -> 5:【後悔】
```

### 体力バッドエンド (`.dvbet`)

体力バッドエンド表を振ります。

#### 例

```
.dvbet
> でたとこサーガ [koi-chan]<体力バッドエンド>: [2, 4] -> 6: 【暴走】
```

### 気力バッドエンド (`.dmbet`)

気力バッドエンド表を振ります。

#### 例

```
.dmbet
> でたとこサーガ [koi-chan]<気力バッドエンド>: [6, 5] -> 11: 【気絶】
```
